## Descrizione del Progetto
Questa repository contiene i materiali per la prova pratica riguardante i fondamenti della virtualizzazione e l'uso degli strumenti di versionamento del codice. 

L'obiettivo dell'esercitazione è illustrare:
1. I concetti di Ipervisore (con focus su VirtualBox come Ipervisore di Tipo 2).
2. L'importanza del flusso di lavoro basato su Git e GitHub per backup, versionamento e creazione di un portfolio digitale.
